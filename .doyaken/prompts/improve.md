# Code Improvement Prompt

Make systematic improvements across the codebase:

## Code Quality Improvements
- Fix any RuboCop violations
- Improve code organization and structure
- Enhance error handling and user feedback
- Add missing tests for critical paths
- Improve test data factories

## User Experience Improvements
- Better error messages (i18n compliant)
- Improved form validation feedback
- Enhanced accessibility (ARIA labels, keyboard navigation)
- Better loading states and feedback
- Improved mobile responsiveness

## Developer Experience
- Better documentation and comments
- Clearer method and variable names
- Improved logging and debugging tools
- Better development environment setup
- Enhanced README and documentation

## Infrastructure Improvements
- Improve database indexes
- Optimize migrations
- Enhance CI/CD pipeline
- Improve monitoring and observability
- Better deployment process

## Technical Debt Reduction
- Remove unused code and dependencies
- Update outdated patterns
- Consolidate similar implementations
- Improve configuration management
- Better environment variable handling

Focus on small, incremental improvements that provide immediate value.
